# imard module repo

To go to the source, install and run the server, use respectively

<pre><code>
cd &lt;sourcename&gt;
npm install
npm start
</code></pre>
<i>Available urls:</i>
<pre><code>
/                   [GET]
/start (equal to /) [GET]
/clone-redirect     [POST]
/clone              [PUT]
/compile            [POST]
</code></pre>
